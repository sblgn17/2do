# 2Do Projekt

 Überblick

Das Projekt **2Do** ist eine Java-basierte Client-Server-Anwendung zur strukturierten Aufgabenverwaltung. Der Fokus liegt auf Mehrbenutzerfähigkeit, klarer Aufgabenorganisation und transparenter Zusammenarbeit.
Funktionaler Umfang

* Anmeldung mehrerer Benutzer mit getrennten Aufgabenbereichen

* Erstellung, Bearbeitung und Nachverfolgung eigener Tasks

* Abonnieren von Tasks anderer Benutzer zur gemeinsamen Übersicht

* Statusbasierte Einteilung aller Tasks in:
  
  * **Offen**
  
  * **In Bearbeitung**
  
  * **Erledigt**

* Kalenderansicht zur Filterung und Darstellung von Tasks nach Datum

Technische Grundlage
--------------------

Die Anwendung wird vollständig in **Java** umgesetzt und folgt einem **Client-Server-Architekturmodell**, um Erweiterbarkeit und saubere Trennung von Logik und Darstellung zu gewährleisten.
